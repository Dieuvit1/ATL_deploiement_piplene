# Pipeline de vérification de news financières

Projet ATL Déploiement Pipeline.

L’objectif est de concevoir un pipeline Big Data permettant de collecter des news en temps réel, de vérifier leur légitimité et de stocker les résultats pour aider à anticiper l’évolution de certains marchés financiers.

## Technologies prévues

- Kafka pour l’ingestion temps réel
- Spark pour le traitement des données
- Hadoop HDFS pour le stockage Big Data
- MongoDB pour la base de données finale
- Python pour les scripts d’extraction et de traitement

## Données collectées

- Titre
- Résumé
- Date d’occurrence de l’évènement
- Date de publication
- Source
- URL
- Statut de vérification
- Score de suspicion

## Objectif pédagogique

Ce projet permet de concevoir, déployer, tester, surveiller et documenter un pipeline de données Big Data.
